Vehicle Wash Booking Management System is simple and rich feature, It can be used for a car wash reservation system Business owners can manage bookings online using rich feature admin panel without any hassles. Washigniter is a fully responsive and easy to use. You can also integrate with any existing website by using this Washigniter.

Admin panel provides a rich feature like add main vehicle category, list of services, booking history, website setting like available time and out of service time, time interval etc.

This system also provides website integration, you can just copy and paste particular script generate in admin panel that will load booking system on your existing website.

Installation

To install, follow the steps bellow:

STEP 1
  1. Extract the archive file on your machine that you've downloaded from CodeCanyon.
  2. Upload your downloaded zip file to your sever using any FTP client or cpanel.
  3. Create a MySql Database via your web hosting providers Control Panel (e.g. cPanel). Once you have created the database, remember the following information
  • Database name
  • Database username
  • Database password
  • Database host
STEP 2

Open your web browser and enter your url

For Example: http://your-domain-name.com/install or if files is in your sub directory enter your url with directory name like http://www.your-domain-name.com/vms/install in the address bar.

The following screen will appear

Fillout the field for database connections

Once installed successfully, automatically login to dashboard page

Note: please find the below default login credentials

  1. Username: admin
  2. Password: admin
